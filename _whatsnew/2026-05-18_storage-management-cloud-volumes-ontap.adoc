=== Déploiement simplifié de Cloud Volumes ONTAP dans Azure

Vous pouvez désormais utiliser la création rapide pour déployer Cloud Volumes ONTAP dans Azure, aussi bien pour les configurations à nœud unique que pour les configurations à haute disponibilité (HA). Ce processus simplifié réduit le nombre d'étapes par rapport à la méthode avancée, définit automatiquement les valeurs par défaut sur une seule page et minimise la navigation, rendant le déploiement plus rapide et plus facile.


NOTE: Si vous utilisez la création rapide pour déployer un nouveau système Cloud Volumes ONTAP dans Azure, la NetApp Console sélectionne par défaut les informations d'identification Azure. Vous pouvez déployer tous vos systèmes Cloud Volumes ONTAP en utilisant les informations d'identification Azure initiales. Si vous souhaitez utiliser un autre abonnement et/ou d'autres informations d'identification, vous devez passer au déploiement en mode avancé. Cette limitation sera levée dans une prochaine version.
