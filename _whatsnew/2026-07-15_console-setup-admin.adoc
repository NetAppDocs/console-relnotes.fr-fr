[[console-agent-4-9-0]]
=== Agent Console 4.9.0

La version 4.9.0 prend en charge à la fois le mode standard et le mode restreint.

Cette version de l'agent Console inclut des améliorations de sécurité et des corrections de bugs.



[[netapp-console-administration]]
=== Administration de la NetApp Console

NetApp Console 4.9.0 administration inclut la nouvelle fonctionnalité suivante.

.Prise en charge du contrôle d'accès basé sur les rôles pour Cloud Volumes ONTAP
La console prend en charge le contrôle d'accès basé sur les rôles pour Cloud Volumes ONTAP.

link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-cvo-roles.html["En savoir plus sur le contrôle d'accès basé sur les rôles pour Cloud Volumes ONTAP"^].
